GoagentUpdate
=============

GoagentUpdate

Goagent自动部署程序

用于自动部署Goagent本地客户端

可以设置开机自动启动

程序请求管理员权限是因为需要自动导入证书



#### ChangeLog ####


* v1.0 release 2013/12/02
       * 自动扫描已经存在的Goagent文件夹
       * 开机自动启动
       * 自动导入证书
       * 自动打包生成自解压rar包
